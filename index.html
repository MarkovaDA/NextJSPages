Next.js Учебный Проект

Этот проект создан как песочница для изучения Next.js App Router. Проходи уроки по порядку и меняй код под свои задачи.

1. Basics: Server Component

Простой маршрут на серверном компоненте и разметка в App Router.

2. Client Component: Counter

Интерактивность через useState и директиву use client.

3. Data Fetching

Загрузка данных на сервере через fetch в page.tsx.

4. API Route

Пример route handler в папке app/api.

Быстрый старт

  1. Запусти проект: npm run dev
  2. Открой любой урок и попробуй изменить код.
  3. После изменений проверь результат в браузере и в API: /api/hello.
